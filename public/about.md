# About

This website displays the annual **ICA conference** data (2003â€“2018),
which contains:

- 27,466 papers  
- 21,038 authors  
- 4,935 sessions  

The data come from the
[official website of ICA](https://www.icahdq.org/page/annual-conference).

The source code of this web app in `React` will be released after ICA 2026 peer-reviewing. 

### API

The API is available at https://ica-cof.vercel.app/api. For example:

- [https://ica-cof.vercel.app/api/papers](https://ica-cof.vercel.app/api/papers)
- [https://ica-cof.vercel.app/api/authors](https://ica-cof.vercel.app/api/authors)
- [https://ica-cof.vercel.app/api/sessions](https://ica-cof.vercel.app/api/sessions)
- [https://ica-cof.vercel.app/api/papers?title_contains=communication&year=2003](https://ica-cof.vercel.app/api/papers?title_contains=communication&year=2003)
